# Images Directory

## Required Images

### Headshot
- **File:** `headshot.jpg`
- **Size:** 300x300px minimum (will be displayed as 150px circle)
- **Format:** JPG or PNG
- **Max file size:** 200KB
- **Tips:** Professional photo, good lighting, solid background

### Company Logos

Add logos to `logos/` directory with these filenames:

- `adobe.png` - Adobe logo
- `faire.png` - Faire logo  
- `intuit.png` - Intuit logo
- `accenture.png` - Accenture logo
- `baml.png` - Bank of America Merrill Lynch logo

**Logo specs:**
- **Height:** 40px (width can vary)
- **Format:** PNG with transparent background preferred
- **Max file size:** 50KB each
- **Style:** Use official brand logos from company press kits

## Where to Get Logos

1. **Company press/media kits** (search "[Company Name] press kit")
2. **Brandfetch.com** - free logo downloads
3. **Company websites** - usually in footer or "About" section

## Image Optimization

Before adding images, optimize them:

**Mac:**
- Use Preview → Export → Reduce file size

**Online tools:**
- TinyPNG.com
- Squoosh.app

Target: All images total <1MB for fast page load.
